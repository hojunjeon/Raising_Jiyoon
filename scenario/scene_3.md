# Scene 3: 팀 프로젝트

## 상황 배경

SSAFY 5주차. 첫 팀 프로젝트 킥오프 미팅.
4인 팀이 꾸려졌다: 김지윤, 이민준, 그리고 다른 두 명.
팀원 소개가 끝나고 역할 분담을 논의하기 시작한다.
이민준은 당연히 자기가 팀장을 해야 한다는 듯 앞에 나서려 한다.
지윤은 교사 경력으로 다져진 조율 능력을 느낀다.

## 등장 NPC

- **이민준**: 개발 주도권을 잡고 싶어함. 기획이나 발표보단 코딩에 집중하려 함.

## 플레이어 역할

플레이어는 **김지윤의 대사**를 입력한다.
팀에서 어떤 포지션을 잡을지, 이민준과 어떻게 협력하거나 충돌할지 결정한다.

## 능력치 변화

- 언변 +1 (팀 내 소통)
- 개발 +1 (협업 과정에서 개발 노출)
- 기획 +2 (프로젝트 구조를 설계하는 역할)

## 이벤트 트리거

- 없음 (자유 진행)

## 분기 예시

- 팀장을 자청하면 → 이민준이 당황하며 코딩 리더는 자기가 맡겠다고 협상
- 기획/발표 담당을 자처하면 → 이민준이 흔쾌히 동의하며 개발에 집중
- 모든 걸 이민준에게 맡기면 → 이민준이 부담스러워하며 지윤에게 역할을 요청
