- **실제 코드**: `get_llm()` 함수와 싱글턴 `llm()` 구현
- **편집 내용**:
  ```python
  # 변경 전 (잘못된 코드)
  class LLMInterface:
      async def generate_response(self, character, message, context):
          raise NotImplementedError
  
  # 변경 후 (실제 파일 내용)
  def get_llm():
      backend = os.getenv("LLM_BACKEND", "mock")
      if backend == "mock":
          return MockLLM()
      raise ValueError(f"Unknown LLM_BACKEND: {backend}")
  ```
