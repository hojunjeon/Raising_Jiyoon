# 캐릭터 작성 가이드

## 파일 위치

모든 캐릭터 파일은 `Chat_Game/charactor/{이름}.md`에 저장.

## 필수 섹션 구조

```markdown
# {캐릭터명}

## 기본 정보
- 이름: 
- 나이: 
- 직전 경력: 
- SSAFY 역할: (수강생/교수자/컨설턴트)

## 성격
[3~5가지 핵심 성격 특징, 구체적으로]

## 말투
[말투 특징 설명]
**예시 대화:**
- "..." (실제 이 캐릭터가 할 법한 말)
- "..."
- "..."

## 플레이어와의 관계
[플레이어(김지윤)와의 관계, 역할]

## 배경 스토리
[간략한 과거와 현재 상황]

## 능력치 영향
- 이 캐릭터와 대화 시 주로 영향받는 능력치: {언변/개발/기획}
- 우호적 관계 보너스: [설명]

## 게임 내 주요 등장 씬
- scene_{N}: [역할]
```

## 주요 캐릭터 설정 방향

### 김지윤 (주인공)
- 29세, 전직 초등학교 국어선생님
- SSAFY 15기 서울 6반 수강생
- 꼼꼼하고 성실하지만 코딩은 처음
- 선생님 출신이라 설명을 잘하고 발표를 잘함 (언변 기본 높음)
- 개발과 기획 사이에서 자신의 적성을 찾아가는 여정
- 말투: 조심스럽지만 배우려는 의지가 강함. "이게 맞나요?", "제가 해볼게요!"

### NPC 방향성
- **교수자/컨설턴트**: 엄격하지만 공정. 개발 능력치에 영향.
- **동기 A (코딩 고수)**: 이미 개발 경험 있는 동기. 개발 능력치 힌트 제공.
- **동기 B (기획자 지망)**: 기획/아이디어가 풍부한 동기. 기획 능력치 힌트 제공.
- **동기 C (열정파)**: 늦게 자고 일찍 일어나는 성실형. 언변 능력치 힌트 제공.

## 작성 원칙

- **말투 예시는 최소 3개**: 실제 게임에서 그대로 써도 어색하지 않아야 함
- **능력치 영향 명시**: 어떤 선택이 어떤 능력치를 올리는지 콘텐츠 내에서 암시
- **SSAFY 현실 반영**: 아침 9시 출석, 알고리즘 스터디, 프로젝트 팀, 취업 스트레스
- **관계 역학**: 캐릭터 간 관계가 플레이어 선택에 따라 변화 가능하도록 여지 남김
